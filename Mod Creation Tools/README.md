# Mod Creation Tools

This is my personal set of tools used for making mods in Bomb Rush

- **BRCMod_Maker:** This is a BAT file that will compile any mod for you. All you need to is drop it in the root of your project, and run it

- **SnailsDebugger:** This is my in-game mod to track FPS and Cords, mainly meant to help with making maps, but it can also be helpful in seeing how well your mods run.
