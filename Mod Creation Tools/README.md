# Mod Creation Tools

This is my personal set of tools used for making mods in Bomb Rush

- **BRCMod_Maker:** This is a BAT file that will compile any mod for you. It needs to be put on the root of your mod project, and then all you have to do is run it to have your mod compile. 

- **SnailsDebugger:** This is my in-game mod to track FPS and Cords, mainly meant to help with making maps, but it can also be helpful in seeing how well your mods run. Both options can be toggled from the in-game phone app from this mod.
